Attendance Website Sidebar Options

  • Dashboard:
    The dashboard is the main page of the website that provides an overview of the attendance data collected.
  • Classes:
    The classes page displays a list of all classes that have been created on the website.
  • New Class:
    The new class option allows the user to create a new class and add it to the list of existing classes.
  • New Student:
    The new student option allows the user to add a new student to the database.
  • FAQ/Help:
    The FAQ/help page provides answers to commonly asked questions and can provide assistance for users who may have questions about how to use the website or any technical issues they may be experiencing.
  • Settings:
    The settings page allows the user to configure various settings for the website.
  • Logout:
    The logout option allows the user to securely log out of their account.
